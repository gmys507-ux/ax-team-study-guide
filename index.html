? 왜 주간회의인가

  • 새 회의를 만들지 마세요. 주간회의 끝에 15~20분만 — 부담 0, 지속 가능.
  • AI는 혼자 쓰면 잊히고, 같이 나누면 번집니다. 한 명의 작은 성공이 팀의 표준이 돼요.
  • 리더가 완벽히 시연할 필요 없어요. "나도 이거 막혔어"가 더 좋은 공유(심리적 안전).
  • 4주 동안 루틴을 만들면 멘토링이 끝나도 팀에 남습니다.

2 이런 걸 공유해요 — 주제별 예시

2주차부터 데이터·대시보드·자동화·에이전트봇 세션이 시작돼요. 거기서 직접 실습하고 만들어본 것을 주간회의에서 팀과 나눠보세요. 아래에서 주제를 골라 예시를 확인하세요 👇

💡 포털에 각 팀이 올려주신 인사이트를 기반으로 AX 스터디를 진행할 계획이에요. 그래서 스터디를 위해 내용을 따로 정리하실 필요가 없어요 — 평소 챌린지에 인증만 해두시면 끝!
📊 데이터 ⚙️ 자동화 🤖 에이전트봇 🟢 클로드코드
📥 데이터 가져오기(API)"사내/공공 API를 클로드코드로 연결해, 매번 손으로 받던 지표를 자동으로 가져왔어요."
🔎 분석"지난 분기 매출 CSV를 주고 '놓친 패턴·이상한 점 5개 찾아줘' → 몰랐던 추세 발견."
📊 시각화"숫자 표를 '한눈에 보이는 차트로' 시켰더니 보고가 훨씬 쉬워졌어요."
📈 대시보드"팀 KPI를 한 페이지 대시보드로 만들어, 매주 자동 업데이트되게 연결."
🔗 팀에 공유"그 대시보드 링크를 팀 전체에 공유 → 모두가 같은 숫자를 보고 회의해요."

데이터를 가져오기 → 분석 → 시각화 → 대시보드 → 팀 공유까지. 한 단계만 해봐도 좋은 공유예요.

🧹 데이터 정리"지저분한 엑셀을 '한 행 = 한 건'으로 자동 정리하는 스크립트를 만들었어요."
🔁 반복 작업"카페24 매출 엑셀 → 구글시트에 자동 입력(버튼 한 번)으로 바꿨어요."
📨 알림·취합"협력업체 진행이 3일 멈추면 담당자에게 슬랙으로 자동 리마인드."
📄 문서 생성"회의록을 넣으면 '액션 아이템 표'로 자동 정리해줘요."
⏱️ 결과"매주 2시간 걸리던 일이 10분으로 — 남은 시간은 기획에 썼어요."

핵심은 "매번 손으로 하던 그 일" 하나를 자동으로. 작게 시작하세요.

🤖 만든 봇"타겟을 넣으면 트렌드·레퍼런스를 모아 노션에 정리해주는 봇을 만들었어요."
🔌 도구 연결(MCP)"노션·슬랙을 AI에 연결(MCP)하니, 말로 시키면 실제 페이지를 만들어줘요."
🧩 역할 나누기"큰 일을 작은 역할로 쪼개 여러 AI(서브에이전트)가 나눠 처리하게 했어요."
💡 알게된 점"봇은 '한 번에 완벽'보다, 작게 만들고 피드백을 쌓을수록 똑똑해져요."
🔗 팀에 공유"내가 만든 봇 사용법을 팀에 공유 → 다른 팀원도 바로 활용해요."

"사람이 매번 챙기던 일"을 봇에게. 작게 만들고 키우기가 정답.

📁 CLAUDE.md"프로젝트에 CLAUDE.md로 '내 맥락'을 적어두니, 매번 설명 안 해도 알아들어요."
🗂️ 폴더 구조"작업을 폴더로 나눠 정리하니 AI가 헷갈리지 않고 정확히 작업해요."
🔁 피드백 누적"결과가 별로여도 '왜 별론지' 피드백을 주면, 쓸수록 결과가 좋아져요."
🧰 스킬·MCP"자주 쓰는 작업을 '스킬'로 저장해두면 다음부터 한 단어로 끝나요."
⚠️ 노하우"한 번에 큰 걸 시키지 말고, 작은 단계로 쪼개 시키면 실패가 확 줄어요."

클로드코드는 '기억(CLAUDE.md)' + '작게 쪼개기'가 핵심 노하우.

4 운영 팁

돌아가며 — 매주 발표자 1명을 미리 지정(다음 주 예고)해 부담을 나눠요.
미리 준비 — 회의 전에 각자 포털 챌린지에 이번 주 인사이트를 인증해 두면, AX 스터디 탭에 자동으로 쌓여서 회의가 짧고 알차져요.
작게 — "엑셀 정리 한 번", "메일 초안 한 번"도 충분한 사례. 거창할 필요 없음.
실패 환영 — "안 됐어요"가 최고의 학습 재료. 리더가 먼저 실패담을 꺼내면 분위기가 열려요.
연결 — 포털 챌린지에 인증하면 포인트·랭킹 자동 적립 + AX 스터디 탭에 팀별 아카이빙까지 한 번에.
